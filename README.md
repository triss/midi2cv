# midi2cv

A lightweight MIDI-to-CV converter for a modular synth.

Takes MIDI input and outputs appropriate control voltages to a modular
system via Expert Sleepers ES-3 modules. Built on top of JACK.

## Goals

- Built on JACK (MIDI in, audio out to the ES-3's ADAT/DC-coupled outputs)
- Minimal, understandable code
- Lightweight
- Support multiple configurations (eventually)
- Easy to calibrate
- Simple Makefile
